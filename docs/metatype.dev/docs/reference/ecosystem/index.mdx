# Ecosystem

## [ghjk](https://github.com/metatypedev/ghjk)

[ghjk](https://github.com/metatypedev/ghjk) /gk/ is a programmable runtime manager and an attempt at a successor for asdf. It powers the development of Metatype on its authors' machines, on the CI and containers.

## [Substantial](https://github.com/zifeo/substantial)

[Substantial](https://github.com/zifeo/substantial) is brokerless durable execution for Python. It serves as a sandbox to develop Metatype workflows engine.

## [Lade](https://github.com/zifeo/lade)

[Lade](https://github.com/zifeo/lade) automatically load secrets from your preferred vault as environment variables, and clear them once your shell command is over. The Meta CLI bundles Lade and use it to load secret seamlessly into your typegates.

## [Whiz](https://github.com/zifeo/whiz)

[Whiz](https://github.com/zifeo/whiz) is a modern DAG/tasks runner for multi-platform monorepos. It enables to develop the core components of the ecosystem together and reloading only the necessary tasks on file change.
