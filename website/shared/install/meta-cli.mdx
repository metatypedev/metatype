You can download the binary executable from
[releases page](https://github.com/metatypedev/metatype/releases/) on GitHub, make it
executable and add it to your `$PATH` or use the method below to automate those steps.

```bash
curl -fsSL https://raw.githubusercontent.com/metatypedev/metatype/main/installer.sh | bash

meta upgrade # to upgrade to the latest version
```
